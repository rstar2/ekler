# For Firebase unctions project the package.json has to have

1. Obligatory - Install "firebase-admin" and "firebase-functions"

```json
"dependencies": {
    "firebase-admin": "xxx",
    "firebase-functions": "xxx"
}
```

2. To specify the NodeJS version 8 or 10

```json
"engines": {
    "node": "10"
}
```
